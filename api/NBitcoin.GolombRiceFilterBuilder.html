Class GolombRiceFilterBuilder

Class for creating Golomb-Rice filters for a given block. It provides methods for building two kind of filters out-of-the-box: Basic Filters and Extenden Filters.

BuildBasicFilter(Block)

Builds the basic filter for a given block.

The basic filter is designed to contain everything that a light client needs to sync a regular Bitcoin wallet. A basic filter MUST contain exactly the following items for each transaction in a block:

  • The outpoint of each input, except for the coinbase transaction
  • The scriptPubKey of each output
  • The txid of the transaction itself
